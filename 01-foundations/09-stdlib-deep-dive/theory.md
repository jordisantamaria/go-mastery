# 09 - Standard Library Deep Dive

> Modulo en progreso. Temas a cubrir:

- net/http: servidor, handlers, middleware, mux
- io: Reader, Writer, composicion
- encoding/json: marshal, unmarshal, struct tags, custom encoders
- context: propagacion, timeouts, cancellation
- os / filepath: archivos, directorios
- flag: CLI arguments
- log/slog: structured logging (Go 1.21+)
- time: duraciones, timers, tickers
- sync: primitivas de concurrencia
- strings / strconv / fmt: manipulacion de texto
